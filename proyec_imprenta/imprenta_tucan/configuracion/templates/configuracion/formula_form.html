Fórmulas del sistema — Referencia

Variables estándar disponibles para cualquier producto gráfico.

Variables generales

Q — Cantidad solicitada R — Rendimiento por pliego M — % Merma (ej: 0.05) C — Cantidad de colores F — Caras impresas (1 o 2) Formas — Formas de impresión At — Área impresa m²/unidad Ct — Consumo tinta gr/m²/color Pp — Precio por pliego Pt — Precio por kg tinta Ppl — Precio por plancha Pb — Precio por kg barniz Cb — Consumo barniz gr/m² Plam — Precio por m² laminado

layers Papel

Pliegos = (Q / R) * (1 + M)

Costo_Papel = Pliegos * Pp

water_drop Tinta

Tinta_kg = (Q * At * Ct * C * F) / 1000

Costo_Tinta = Tinta_kg * Pt

view_array Planchas

Planchas = C * F * Formas

Costo_Planchas = Planchas * Ppl

opacity Barniz (opcional)

Barniz_kg = (Q * At * Cb) / 1000

Costo_Barniz = Barniz_kg * Pb

flip Laminado (opcional)

Costo_Laminado = Q * At * Plam

calculate Costo Directo Total

CD = Costo_Papel + Costo_Tinta

+ Costo_Planchas + Costo_Barniz

+ Costo_Laminado

Costo_Unitario = CD / Q

Fórmula compacta para copiar:

((Q/R)*(1+M)*Pp) + ((Q*At*Ct*C*F)/1000*Pt) + (C*F*Formas*Ppl) + ((Q*At*Cb)/1000*Pb) + (Q*At*Plam)
