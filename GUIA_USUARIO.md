# Guía de Usuario - Proyecto Imprenta Tucán

## Introducción
Este sistema permite gestionar clientes, proveedores, productos, pedidos y automatizaciones inteligentes para una imprenta.

## Acceso al sistema
- Ingresa a la web local tras iniciar el servidor (por defecto: http://127.0.0.1:8000/).
- Accede al panel de administración en `/admin` con tu usuario y contraseña de administrador.

## Funcionalidades principales
- Gestión de clientes, productos, proveedores y pedidos.
- Automatización de órdenes sugeridas y ofertas.
- Ranking de clientes y proveedores.
- Notificaciones automáticas por email y portal.
- API REST para integración con otros sistemas.

## Cómo crear un pedido
1. Ingresa al panel de administración o a la sección de pedidos.
2. Completa los datos requeridos (cliente, productos, cantidades, etc.).
3. Guarda el pedido. El sistema puede sugerir órdenes o proveedores automáticamente.

## Automatizaciones inteligentes
- El sistema predice demanda y anticipa compras de insumos críticos.
- Genera ofertas y órdenes sugeridas automáticamente.
- Envía alertas y notificaciones según reglas configuradas.

## Preguntas frecuentes
- **¿Cómo recupero mi contraseña?**
  - Usa la opción "¿Olvidaste tu contraseña?" en la pantalla de login.
- **¿Cómo agrego un nuevo usuario?**
  - Desde el panel de administración, sección Usuarios.
- **¿Cómo consulto el ranking de proveedores?**
  - En la sección de proveedores, consulta la columna "Ranking cargado".

## Soporte
Para dudas o problemas, contacta al administrador del sistema o revisa la documentación técnica incluida en el proyecto.
